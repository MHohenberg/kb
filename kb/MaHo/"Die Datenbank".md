#storyIdee

Eine Gruppe Entdecker trifft auf einen Planeten, der auf erstaunliche Weise eintönig ist. Es ist offensichtlich, dass dort einmal intelligentes Leben existiert haben muss. Auf dem Planeten finden Sie einen großen Komplex von Computern. Diese Computer stellen das Gehirn der Datenbank da: einem System, dass offenbar alle Informationen zum aktuellen Stand des Planeten durchsuchbar vorhält.

Kaum landen Sie auf dem Planeten, treffen Sie auf einen von fünf so genannten Agenten, das sind Roboter und Programme, die die Datenbank aktuell halten. Diese Agent fast alle Daten zu dem Schiff und den Personen. Es gelingt den Personen, ein Interface der Datenbank zu finden: sie fangen an, Anfragen zu stellen, und sind zunächst von der Datenbank beeindruckt. Dann jedoch stellen Sie fest, dass die Datenbank es sich einfach macht, und Informationsanfragen stark vereinfacht, teilweise so stark, dass sie sinnentleert werden.

Während des gesamten Vorgangs stellen Sie fest, dass die Datenbank auch in ihrer physikalischen Repräsentation in einem permanenten Wandel ist: teilweise erscheint es, als ob komplette Bereiche gezielt vernichtet werden.

Über mehrere Stufen finden Sie heraus, dass die Datenbank gezielt versucht, die eigenen Strukturen zu vereinfachen: einfachere Strukturen bedeutet schnellere Suche. Das tut sie unter anderem auch damit, dass sie gezielt reale, physikalisch Objekt vernichtet, die in offensichtlicher Weise von dem Datenbankobjekt abweicht: zum Beispiel versucht der Computer, dass eine weibliche Crew Mitglied zu eliminieren, weil dadurch das Merkmal männlich, weiblich keine Bedeutung mehr hat.
